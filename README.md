# Crystal Math

A Vite Vanilla project showcasing Tailwind CSS for a sleek, responsive web interface with a gradient background.

## Features

- **Responsive Design**: Adjusts seamlessly across different screen sizes.
- **Sleek Interface**: Utilizes Tailwind CSS for modern styling.
- **Interactive Elements**: Includes buttons and dynamic effects.
## Learning Tailwind CSS

This project is a aimed to learn Tailwind CSS and its utility-first approach.
